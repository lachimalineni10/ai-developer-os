# AI Developer OS

> **A production-grade AI Developer Platform built using modern Backend, AI, and DevOps technologies following real-world SaaS engineering practices.**

## 🚀 Vision

AI Developer OS is my flagship engineering project and learning platform. The goal is to design, build, deploy, and maintain a production-ready AI system while following the software engineering practices used in modern SaaS companies.

This project is built with a strong focus on:

* Clean Architecture
* Backend Engineering
* AI Engineering
* DevOps
* Scalability
* Security
* Observability
* Production Readiness

---

## 🎯 Project Goals

* Build reusable AI-powered developer tools.
* Learn production-grade AI system design.
* Practice real-world backend architecture.
* Implement CI/CD and cloud deployment.
* Maintain engineering documentation throughout the project.

## 🛠️ Technology Stack

| Layer            | Technology                  |
| ---------------- | --------------------------- |
| Backend          | FastAPI                     |
| Language         | Python                      |
| Database         | PostgreSQL                  |
| Cache            | Redis                       |
| Queue            | RabbitMQ                    |
| ORM              | SQLAlchemy                  |
| AI               | Gemini / OpenAI / Ollama    |
| Embeddings       | Sentence Transformers       |
| Vector Database  | FAISS → ChromaDB → pgvector |
| Containerization | Docker                      |
| CI/CD            | GitHub Actions              |
| Cloud            | AWS                         |
